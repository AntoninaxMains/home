│  步驟 2: 打開 Chrome 擴充功能頁面                                        │
└──────────────────────────────────────────────────────────────────────────┘

║  常見問題排解                                                            ║
╚══════════════════════════════════════════════════════════════════════════╝

❌ 問題：找不到「載入未封裝項目」按鈕
   ✅ 解決：確保已開啟右上角的「開發人員模式」開關

❌ 問題：載入時出現「Manifest file is missing or unreadable」錯誤
   ✅ 解決：確保選擇的資料夾中包含 manifest.json 文件

❌ 問題：出現「Could not load icon」錯誤
   ✅ 解決：使用 create-icons.html 生成圖示文件

❌ 問題：新標籤頁沒有變化
   ✅ 解決：
      1. 確認擴充功能已啟用（開關是開啟狀態）
      2. 重新啟動 Chrome 瀏覽器
      3. 檢查是否有其他新標籤頁擴充功能衝突

❌ 問題：頁面顯示空白
   ✅ 解決：
      1. 按 F12 打開開發者工具查看錯誤訊息
      2. 確認所有文件（assets、i18n 資料夾）都在
      3. 在 chrome://extensions/ 點擊刷新圖示

║  進階功能                                                                ║
╚══════════════════════════════════════════════════════════════════════════╝

📌 設定為主頁：
   Chrome 設定 → 外觀 → 顯示首頁按鈕
   → 輸入：chrome-extension://[ID]/index.html

📌 在無痕模式中使用：
   擴充功能詳細資料頁面 → 開啟「允許在無痕模式中」

   chrome://extensions/shortcuts
