## ✨ 主な機能**ダークモード、ブックマーク管理、多言語サポートを備えた美しくカスタマイズ可能なブラウザスタートページ。**



- 🚀 **超高速起動** - すべてのリソースをローカル読み込み、待ち時間ゼロ[🌐 ライブデモ](https://home.zakk.au) • [📖 ドキュメント](#-機能) • [🌍 言語](#-多言語対応)

- 🔍 **スマート検索** - Google、Bing、DuckDuckGo、Baidu 対応、リアルタイムサジェスト

- 🎨 **適応型テーマ** - ダーク/ライトモード自動切替、カスタム背景とフィルター**他の言語で読む:** [English](README.en.md) | [简体中文](README.zh-CN.md) | [正體中文](README.zh-TW.md)

- 📱 **レスポンシブデザイン** - デスクトップ、タブレット、スマートフォンに完全対応

- 🌦️ **天気ウィジェット** - Open-Meteo API 統合、Microsoft Fluent Emoji アニメーション</div>

- 🔖 **ブックマーク管理** - カテゴリ整理、カスタムアイコン、ドラッグソート

- 🌍 **多言語対応** - 英語、簡体字中国語、繁体字中国語、日本語内蔵---

- 💾 **ローカルストレージ** - すべてのデータをブラウザに保存、バックエンド不要

### 🎨 ビジュアルカスタマイズ

## 🎯 機能ハイライト- **複数の背景**：グラデーションプリセット、カスタム画像、または単色背景

- **ダークモード**：深度調整可能なエレガントなダークテーマ

### 検索とサジェスト- **背景ぼかし**：カスタマイズ可能なぼかし効果と不透明度コントロール

- 複数検索エンジン切替（Google/Bing/DuckDuckGo/Baidu/カスタム）- **背景フィルター**：白または黒のオーバーレイフィルターを追加

- JSONP リアルタイムサジェスト、多言語対応- **スムーズアニメーション**：インターフェイス全体の流れるようなトランジションとマイクロインタラクション

- スマート URL 検出と直接ナビゲーション

- 検索履歴追跡### 🔖 ブックマーク管理

- **カテゴリ整理**：ブックマークを整理するためのカスタムカテゴリを作成

### ブックマークシステム- **アイコンサポート**：絵文字、カスタム画像URL、またはサイトファビコンの自動取得を使用

- カード式カテゴリ管理- **Simple Icons統合**：数千のブランドロゴから検索して選択

- Emoji、画像URL、自動取得 Favicon 対応- **クイックアクション**：ホバー時にブックマークを編集・削除

- 簡単ドラッグ&ドロップソート- **ドラッグ＆ドロップ**：（近日公開予定）簡単にブックマークを並び替え

- 素早い追加と編集

### 🔍 スマート検索

### 外観カスタマイズ- **複数の検索エンジン**：Google、Bing、DuckDuckGo、Baidu、またはカスタムエンジン

- ダーク/ライトテーマ切替- **検索候補**：入力時にリアルタイムで候補を表示

- カスタム背景（グラデーション/画像/単色）- **検索履歴**：最近の検索を追跡

- 背景ぼかしとフィルター調整- **URL直接開く**：URLを自動検出して直接開く

- アニメーション効果コントロール- **キーボードショートカット**：Enterキーで素早く検索
