## ✨ 機能

### 🎨 ビジュアルカスタマイズ
- **複数の背景**：グラデーションプリセット、カスタム画像、または単色背景
- **ダークモード**：深度調整可能なエレガントなダークテーマ
- **背景ぼかし**：カスタマイズ可能なぼかし効果と不透明度コントロール
- **背景フィルター**：白または黒のオーバーレイフィルターを追加
- **スムーズアニメーション**：インターフェイス全体の流れるようなトランジションとマイクロインタラクション

### 🔖 ブックマーク管理
- **カテゴリ整理**：ブックマークを整理するためのカスタムカテゴリを作成
- **アイコンサポート**：絵文字、カスタム画像URL、またはサイトファビコンの自動取得を使用
- **Simple Icons統合**：数千のブランドロゴから検索して選択
- **クイックアクション**：ホバー時にブックマークを編集・削除
- **ドラッグ＆ドロップ**：（近日公開予定）簡単にブックマークを並び替え

### 🔍 スマート検索
- **複数の検索エンジン**：Google、Bing、DuckDuckGo、Baidu、またはカスタムエンジン
- **検索候補**：入力時にリアルタイムで候補を表示
- **検索履歴**：最近の検索を追跡
- **URL直接開く**：URLを自動検出して直接開く
- **キーボードショートカット**：Enterキーで素早く検索

### 🌍 多言語対応
- **English** (en) - 英語
- **简体中文** (zh-CN) - 簡体字中国語
- **正體中文** (zh-TW) - 繁体字中国語
- **日本語** (ja) - 日本語
- ブラウザの言語に基づく自動検出
- 設定から簡単に言語を切り替え

### 📱 レスポンシブデザイン
- **デスクトップ＆モバイル**：すべての画面サイズに完全対応したレスポンシブレイアウト
- **タッチ最適化**：モバイルデバイスでのスムーズな操作
- **FABメニュー**：モバイルでのクイックアクセス用フローティングアクションボタン
- **適応型UI**：インターフェース要素が画面サイズに自動で調整

### 💾 データの永続化
- **ローカルストレージ**：すべてのデータはブラウザに保存
- **サーバー不要**：完全に静的で、オフラインでも動作
- **インポート/エクスポート**：（近日公開予定）設定のバックアップと復元
- **プライバシー優先**：データ収集や外部追跡なし

## 🚀 クイックスタート

### 方法1：直接使用
1. [https://home.zakk.au](https://home.zakk.au) にアクセス
2. ブラウザのスタートページに設定
3. カスタマイズを開始！

### 方法2：セルフホスティング
```bash
# リポジトリをクローン
git clone https://github.com/AntoninaxMains/home.git

# ディレクトリに移動
cd home

# ブラウザで index.html を開く
# または任意の静的ファイルサーバーで提供
python -m http.server 8000
# その後 http://localhost:8000 にアクセス
```

### 方法3：GitHub Pages
1. このリポジトリをフォーク
2. リポジトリ設定でGitHub Pagesを有効化
3. ソースを `main` ブランチに設定
4. スタートページは `https://yourusername.github.io/home` で利用可能

## 🎯 使用ガイド

### ブックマークの追加
1. 右上の **+ ブックマークを追加** ボタンをクリック
2. ブックマークの詳細を入力：
   - **名前**：ブックマークの表示名
   - **URL**：ウェブサイトのアドレス
   - **アイコン**：絵文字、画像URL、または自動取得を使用
   - **カテゴリ**：既存のカテゴリを選択するか新規作成
3. **保存** をクリック

### カテゴリの管理
1. 右上の **フォルダ** アイコンをクリック
2. カスタム名とアイコンで新しいカテゴリを作成
3. 既存のカテゴリの名前変更または削除
4. 削除されたカテゴリのブックマークはメインリストに戻る

### 外観のカスタマイズ
1. **設定** アイコンをクリック
2. お好みのものを選択：
   - **背景タイプ**：グラデーション、画像、または単色
   - **ダークモード**：切り替えて強度を調整
   - **ぼかし効果**：有効にしてぼかしの深さを調整
   - **オーバーレイフィルター**：白または黒のフィルターレイヤーを追加

### 言語の変更
- **デスクトップ**：右上の地球アイコンをクリック
- **モバイル**：FABメニューを開いて言語オプションを選択
- **設定**：4つのサポート言語から選択
