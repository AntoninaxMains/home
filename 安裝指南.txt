  Chrome 擴充功能 - 我的起始頁 - 快速安裝指南
═══════════════════════════════════════════════════════════

📦 步驟 1: 生成圖示
────────────────────────────────────────────────────────────
1. 用瀏覽器打開 create-icons.html
2. 點擊「下載所有圖示」
3. 確保三個 PNG 圖示（icon16.png, icon48.png, icon128.png）
   都在專案根目錄

🔧 步驟 2: 安裝到 Chrome
────────────────────────────────────────────────────────────
1. 打開 Chrome 瀏覽器
2. 輸入網址：chrome://extensions/
3. 開啟右上角的「開發人員模式」
4. 點擊「載入未封裝項目」
5. 選擇本專案資料夾
6. 完成！

✅ 步驟 3: 測試
────────────────────────────────────────────────────────────
1. 開啟新標籤頁（Ctrl+T 或 Cmd+T）
2. 應該會看到您的自訂起始頁
3. 如果沒有顯示，請檢查：
   - 是否正確開啟「開發人員模式」
   - 資料夾中是否有 manifest.json 文件
   - 是否有生成圖示文件

🎨 可選：設定為主頁
────────────────────────────────────────────────────────────
1. Chrome 設定 → 外觀
2. 啟用「顯示首頁按鈕」
3. 選擇「輸入自訂網址」
4. 輸入：chrome-extension://[擴充功能ID]/index.html
   （擴充功能 ID 在 chrome://extensions/ 頁面可以找到）

🔄 更新擴充功能
────────────────────────────────────────────────────────────
修改文件後：
1. 前往 chrome://extensions/
2. 點擊擴充功能卡片上的刷新圖示 🔄

❓ 遇到問題？
────────────────────────────────────────────────────────────
- 查看瀏覽器控制台（F12）是否有錯誤訊息
- 確認所有必需文件都在正確位置
- 閱讀 CHROME_EXTENSION_README.md 獲取詳細說明
